# Video Asset Placeholder

Place your video files in this directory:
- video-novo.webm (WebM format for better compression)
- video-novo.mp4 (MP4 format for broader compatibility)

Recommended video specifications:
- Resolution: 1920x1080 or 1280x720
- Frame rate: 24-30 fps
- Bitrate: 2-5 Mbps for web
- Codec: H.264 for MP4, VP9 for WebM
- Keep file size under 10MB for better loading performance
